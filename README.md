# Trader Sentiment Analysis – Hyperliquid

This project analyzes how Bitcoin market sentiment (Fear & Greed Index) impacts trader performance using historical Hyperliquid trading data.

## Key Findings:
- Extreme Greed periods generate the highest average profit but show high volatility.
- Fear periods show strong win rates and disciplined trading behavior.
- Greed periods show increased losses, suggesting emotional overconfidence.
- Neutral sentiment is more stable with moderate performance.
- Market sentiment significantly influences risk-taking and profitability.
